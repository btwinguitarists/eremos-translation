# OT Warfare / Ḥerem / Apostasy-Punishment Ethics — Translation Policy

**Status:** LOCKED 2026-05-15
**Triggered by:** Numbers end-of-book audit, Item C (NUM 25:6-9 + 31:17-18)
**Cross-AI consensus:** ChatGPT, Gemini, Logos AI 2026-05-15
**Scope:** Pentateuch + Joshua + Judges + Samuel + Kings + Chronicles (any text involving ḥerem warfare, capital-punishment apostasy judgments, or zealot executions)

---

## 1. The translation policy

**The Thai surface stays unsanitized.** We translate what the Hebrew says, including:
- Ḥerem (חרם) language — devotion to total destruction
- Capital-punishment apostasy provisions
- Phinehas-type zealot executions (NUM 25:7-8)
- Captive-virgin preservation provisions (NUM 31:17-18)
- Holy-war military narratives (Deut 7, Deut 20, Josh 6-12)

We **do not soften, paraphrase around, or euphemize** the textual content. The standard English translations (ESV, NIV, NRSV, NASB, LEB) do not soften these passages — they render them in equivalent English euphemism where Hebrew uses euphemism (e.g., יָדַע מִשְׁכַּב זָכָר → "had relations with a man" / "lain with a man"), but the violent/judicial register is preserved.

**Justification:** A translation that softens these passages is no longer a translation. The reader's interpretive work is supported by Layer-2 footers and external commentary tools, not by editorial omission.

## 2. Layer-2 footer policy

For verses involving:
- Ḥerem-with-population (NUM 31:17-18, Deut 7:1-2, 1 Sam 15:3)
- Capital-punishment for apostasy/idolatry (Exod 32:27-28, NUM 25:7-8, Deut 13:6-10)
- Captive-marriage provisions (NUM 31:18, Deut 21:10-14)

Add a **restrained Layer-2 footer** that:
1. Acknowledges the morally/theologically difficult content
2. Distinguishes textual translation from moral endorsement
3. Points to this doc as the project's editorial stance
4. **Does NOT** moralize, apologize, defend, or attack the text

**Template footer:**
> **LAYER-2 ({CONTEXT} ETHICS)**: ข้อความนี้เป็นส่วนหนึ่งของ {SPECIFIC-INSTITUTION เช่น "กฎḥerem"} ในบริบทเฉพาะของ {SPECIFIC-EVENT}. ฉบับเอเรโมสรักษาข้อความตามต้นฉบับ ไม่ทำให้นุ่มลง. ดู docs/translator_decisions/ot_warfare_ethics_2026-05.md เรื่องการตีความและจุดยืนของฉบับ.

## 3. What this doc explicitly does NOT do

- Does **not** offer a theological resolution of "Why did God command this?"
- Does **not** rank these texts on moral acceptability
- Does **not** recommend which commentary the reader should consult
- Does **not** claim the texts are easy or comfortable

Those are interpretive moves that belong to study Bibles, commentaries, and pastors — **not** to the Eremos Thai translation surface.

## 4. Distinctions to maintain

| Category | Hebrew marker | Treatment |
|---|---|---|
| Ḥerem-with-population (ban) | חרם hiphil + population direct object | Translate fully + Layer-2 footer |
| Battle-narrative slaughter | הִכָּה לְפִי־חֶרֶב | Translate fully + Layer-2 footer if context allows misreading as endorsement |
| Sex-euphemism for sexual experience | יָדַע מִשְׁכַּב זָכָר | Use Thai euphemism that preserves register (e.g., "ที่ได้รู้จักการนอนกับชาย") |
| Zealot execution | רצח / נקם in priestly/judicial context | Translate as judicial-killing terms; do NOT use the modern Thai term for "assassinate" |
| Apostasy capital-punishment | מוֹת יוּמַת for idolatry/blasphemy | Translate fully + Layer-2 footer at first systematic occurrence in book |
