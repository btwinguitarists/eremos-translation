# Christ-hymn kenosis vocabulary cluster (Philippians 2:5–11)

**Decision date:** 2026-05-02 (formalized after PHP end-of-book external AI review)
**Anchor passage:** Philippians 2:5–11
**Status:** LOCK for PHP. Forward passages (Col 1:15–20, 1 Tim 3:16, Heb 1:1–4, Eph 1:20–23, 1 Pet 3:18–22, Rev 5:9–14) "to be integrated, not pre-decided" — when each ships, validate that this lock generalizes; if it doesn't, surface for re-examination rather than mechanically apply.

## What this doc locks

The lemma-pair distinctions in the Christ-hymn carry the Christological-doctrinal weight; collapsing them in Thai would flatten Pauline theology. The principled mapping is:

| Greek | Thai | Editorial principle |
|---|---|---|
| ἐν **μορφῇ θεοῦ** ὑπάρχων (2:6) | ผู้ทรงดำรงอยู่ใน**สภาพของพระเจ้า** | μορφή = essential-nature (NOT outward appearance — Paul reserves σχῆμα for that) |
| οὐχ **ἁρπαγμὸν** ἡγήσατο τὸ εἶναι ἴσα θεῷ (2:6) | มิได้ทรงถือว่าการเป็นเสมอกับพระเจ้าเป็น**สิ่งที่ต้องยึดไว้** | res rapta — Christ already possessed equality, chose not to selfishly cling |
| ἑαυτὸν **ἐκένωσεν** (2:7) | ทรง**สละพระองค์เอง** | NOT "poured-out divinity" (kenotic-heresy); voluntary self-renunciation of privilege |
| **μορφὴν δούλου** λαβών (2:7) | ทรงรับ**สภาพของทาส** | Same μορφή lemma → same สภาพ rendering, preserving Pauline parallel |
| ἐν **ὁμοιώματι** ἀνθρώπων γενόμενος (2:7) | ทรงบังเกิดใน**ลักษณะ**ของมนุษย์ | Real-likeness, NOT mere docetic-appearance |
| **σχήματι** εὑρεθεὶς ὡς ἄνθρωπος (2:7) | เมื่อทรงปรากฏใน**รูป**ของมนุษย์ | σχῆμα = outward-form, distinct from μορφή's essential-nature |
| **ἐταπείνωσεν ἑαυτόν** ... **θανάτου σταυροῦ** (2:8) | ทรง**ถ่อมพระองค์**เองลง ... **ใช่ ความตายบนกางเขน** | Cognate verbal-link with v.3 ταπεινοφροσύνη; emphatic δέ → "ใช่" |
| διὸ καὶ ὁ θεὸς αὐτὸν **ὑπερύψωσεν** (2:9) | เพราะเหตุนี้ พระเจ้าจึงทรง**เทิดทูนพระองค์ขึ้นสู่ที่สูงสุด** | NT-HAPAX ὑπέρ-intensified verb; preserves intensive-superlative force |
| πᾶν γόνυ κάμψῃ ... πᾶσα γλῶσσα ἐξομολογήσηται (2:10–11) | ทุกหัวเข่าจะคุกลง ... ทุกลิ้นจะสารภาพยอมรับ | OT-allusion Isa 45:23 LXX (YHWH-self-reference applied to Christ) |
| **κύριος Ἰησοῦς Χριστός** (2:11) | พระเยซูคริสต์ทรง**เป็นองค์พระผู้เป็นเจ้า** | LXX-divine-name applied to Jesus; corpus-locked κύριος → องค์พระผู้เป็นเจ้า |

## Why this is doctrinally load-bearing

Two heresies must be avoided simultaneously:

- **Docetic** (Christ only *appeared* human): blocked by ὁμοίωμα → ลักษณะ ("real-likeness") and σχῆμα → รูป with "ปรากฏ" ("appeared in form"). The Thai keeps the human form genuine, not phantasmal.
- **Kenotic** (Christ surrendered divine essence): blocked by ἐκένωσεν → สละ ("renounced/gave up") rather than "emptied out divinity," and by μορφή → สภาพ uniformly across "of God" + "of a slave" — same essential-nature lemma, opposite social positioning.

The Thai-evangelical Protestant orthodoxy here is conservative: pre-existence + self-renunciation of privilege + genuine humanity + post-resurrection exaltation, all without compromising divine nature.

## Colossians cosmic-Christ hymn precedent (1:15–20) — added 2026-05-02

The COL hymn is a **distinct vocabulary cluster** under the same Christological-hymn umbrella as PHP 2:5–11. PHP encodes **kenotic** Christology (incarnation + self-renunciation + exaltation); COL encodes **cosmic** Christology (image + agency-in-creation + reconciliation-of-all-things). Different lexicon, different theological frame — both orthodox, both anchored.

Per all three external AI reviewers (Claude + Gemini + ChatGPT) on the COL end-of-book review (Item A), this sub-section is **precedent-level documentation, not a final cross-corpus lock.** The lock generalizes when validated against Eph 1:20–23 / Heb 1:1–4 / 1 Pet 3:18–22 / Rev 5:9–14 as those passages ship.

| Greek | Thai | Editorial principle |
|---|---|---|
| εἰκὼν τοῦ θεοῦ τοῦ ἀοράτου (1:15) | **ฉายา**ของพระเจ้าผู้ไม่ทรงปรากฏ | εἰκών = revelatory-image; preserves "the Son makes the invisible Father known" without flattening to mere visual likeness |
| **πρωτότοκος πάσης κτίσεως** (1:15) | **บุตรหัวปีเหนือ**สิ่งที่ทรงสร้างทั้งสิ้น | rank-of-preeminence; เหนือ blocks the partitive Arian misreading that COL 1:16 itself excludes (see `prototokos_pauline_2026-05.md`) |
| ἐν αὐτῷ ἐκτίσθη τὰ πάντα (1:16) | ในพระองค์ ทุกสิ่งได้ทรงถูกสร้างขึ้น | ἐν αὐτῷ retained as agent-of-creation; the four-tier spiritual-beings hierarchy follows (see `spiritual_beings_hierarchy_2026-05.md`) |
| τὰ πάντα ἐν αὐτῷ συνέστηκεν (1:17) | ในพระองค์ ทุกสิ่งดำรงอยู่ด้วยกัน | συνίστημι = "hold together"; ดำรงอยู่ด้วยกัน preserves the active sustaining force without doctrinal-paraphrase |

| **πᾶν τὸ πλήρωμα** (1:19) | **ความบริบูรณ์ทั้งสิ้น** | πλήρωμα blocks gnostic-emanation distribution at 1:19; "all the fullness" remains undivided in Christ |
| **ἀποκαταλλάξαι τὰ πάντα εἰς αὐτόν** (1:20) | **ให้ทุกสิ่งคืนดี**กับพระองค์ | reconciliation, not universalist salvation — the cross-blood means + Pauline soteriological frame contain it (verse-level KD names this) |
| **εἰρηνοποιήσας** διὰ τοῦ αἵματος τοῦ σταυροῦ αὐτοῦ (1:20) | **ทำให้เกิดสันติสุข** โดยพระโลหิตของพระองค์บนกางเขน | NT-HAPAX εἰρηνοποιέω; cross-blood means = pivotal anti-universalist constraint |
| **ἀποκατήλλαξεν ἐν τῷ σώματι τῆς σαρκὸς αὐτοῦ διὰ τοῦ θανάτου** (1:22) | ทรงให้คืนดี ในพระวรกายฝ่ายเนื้อหนังของพระองค์โดยทางความตาย | royal-prefix **พระวรกาย** for Christ's pre-resurrection body — see `sarx_pauline_2026-05.md` for the σάρξ corpus rule |

## Two clusters under one umbrella

The doc deliberately presents PHP and COL as **two separate vocabulary clusters under one Christological-hymn umbrella** (per Claude's framing). They share:

- An anti-docetic + anti-Arian + anti-kenotic-heresy posture
- Royal-register treatment of Christ's bodily existence (PHP 2:7 σχῆμα → รูป + COL 1:22 σώμα τῆς σαρκός → พระวรกายฝ่ายเนื้อหนัง)
- Anchor-passage status for downstream high-Christology

They differ:

- PHP frames the hymn's center as **self-renunciation** (kenosis); COL frames it as **agency** (creation + reconciliation)
- PHP's μορφή / σχῆμα contrast doesn't appear in COL; COL's εἰκών / πλήρωμα don't appear in PHP
- PHP exaltation = ὑπερύψωσεν (lift-above); COL exaltation = πρωτεύων (rank-be-first)

When forward passages (Eph 1:20–23, Heb 1:1–4, 1 Pet 3:18–22, Rev 5:9–14) ship, **validate against both clusters separately**, not as a single fused vocabulary set. If a forward passage echoes PHP's kenotic frame, follow PHP's lock. If it echoes COL's cosmic frame, follow COL's lock. If it merges the two (Eph 1:20–23 is the most likely candidate), surface for explicit cross-corpus auditing rather than mechanically applying.
