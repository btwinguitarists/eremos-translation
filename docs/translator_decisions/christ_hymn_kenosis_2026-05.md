# Christ-hymn kenosis vocabulary cluster (Philippians 2:5–11)

**Decision date:** 2026-05-02 (formalized after PHP end-of-book external AI review)

**Status:** LOCK for PHP. Forward passages (Col 1:15–20, 1 Tim 3:16, Heb 1:1–4, Eph 1:20–23, 1 Pet 3:18–22, Rev 5:9–14) "to be integrated, not pre-decided" — when each ships, validate that this lock generalizes; if it doesn't, surface for re-examination rather than mechanically apply.

## What this doc locks

The lemma-pair distinctions in the Christ-hymn carry the Christological-doctrinal weight; collapsing them in Thai would flatten Pauline theology. The principled mapping is:

| Greek | Thai | Editorial principle |
|---|---|---|
| ἐν **μορφῇ θεοῦ** ὑπάρχων (2:6) | ผู้ทรงดำรงอยู่ใน**สภาพของพระเจ้า** | μορφή = essential-nature (NOT outward appearance — Paul reserves σχῆμα for that) |
| οὐχ **ἁρπαγμὸν** ἡγήσατο τὸ εἶναι ἴσα θεῷ (2:6) | มิได้ทรงถือว่าการเป็นเสมอกับพระเจ้าเป็น**สิ่งที่ต้องยึดไว้** | res rapta — Christ already possessed equality, chose not to selfishly cling |
| ἑαυτὸν **ἐκένωσεν** (2:7) | ทรง**สละพระองค์เอง** | NOT "poured-out divinity" (kenotic-heresy); voluntary self-renunciation of privilege |
| **μορφὴν δούλου** λαβών (2:7) | ทรงรับ**สภาพของทาส** | Same μορφή lemma → same สภาพ rendering, preserving Pauline parallel |
| ἐν **ὁμοιώματι** ἀνθρώπων γενόμενος (2:7) | ทรงบังเกิดใน**ลักษณะ**ของมนุษย์ | Real-likeness, NOT mere docetic-appearance |
| **σχήματι** εὑρεθεὶς ὡς ἄνθρωπος (2:7) | เมื่อทรงปรากฏใน**รูป**ของมนุษย์ | σχῆμα = outward-form, distinct from μορφή's essential-nature |
| **ἐταπείνωσεν ἑαυτόν** ... **θανάτου σταυροῦ** (2:8) | ทรง**ถ่อมพระองค์**เองลง ... **ใช่ ความตายบนกางเขน** | Cognate verbal-link with v.3 ταπεινοφροσύνη; emphatic δέ → "ใช่" |
| διὸ καὶ ὁ θεὸς αὐτὸν **ὑπερύψωσεν** (2:9) | เพราะเหตุนี้ พระเจ้าจึงทรง**เทิดทูนพระองค์ขึ้นสู่ที่สูงสุด** | NT-HAPAX ὑπέρ-intensified verb; preserves intensive-superlative force |
| πᾶν γόνυ κάμψῃ ... πᾶσα γλῶσσα ἐξομολογήσηται (2:10–11) | ทุกหัวเข่าจะคุกลง ... ทุกลิ้นจะสารภาพยอมรับ | OT-allusion Isa 45:23 LXX (YHWH-self-reference applied to Christ) |
| **κύριος Ἰησοῦς Χριστός** (2:11) | พระเยซูคริสต์ทรง**เป็นองค์พระผู้เป็นเจ้า** | LXX-divine-name applied to Jesus; corpus-locked κύριος → องค์พระผู้เป็นเจ้า |

## Why this is doctrinally load-bearing

Two heresies must be avoided simultaneously:

- **Docetic** (Christ only *appeared* human): blocked by ὁμοίωμα → ลักษณะ ("real-likeness") and σχῆμα → รูป with "ปรากฏ" ("appeared in form"). The Thai keeps the human form genuine, not phantasmal.
- **Kenotic** (Christ surrendered divine essence): blocked by ἐκένωσεν → สละ ("renounced/gave up") rather than "emptied out divinity," and by μορφή → สภาพ uniformly across "of God" + "of a slave" — same essential-nature lemma, opposite social positioning.

The Thai-evangelical Protestant orthodoxy here is conservative: pre-existence + self-renunciation of privilege + genuine humanity + post-resurrection exaltation, all without compromising divine nature.

## Forward-facing parallels (DO NOT pre-lock)

When each of these ships, validate against this passage but don't mechanically import the vocabulary:

- **Col 1:15–20** — εἰκὼν τοῦ θεοῦ τοῦ ἀοράτου / πρωτότοκος πάσης κτίσεως / πλήρωμα. Different lexicon (no μορφή), different theological frame (cosmic Christology vs functional kenosis). ChatGPT's caution: "validate the lock, not be governed by it pre-emptively."
- **1 Tim 3:16** — ἐφανερώθη ἐν σαρκί. Hymn fragment; "manifested in flesh" is the relevant clue.
- **Heb 1:1–4** — χαρακτὴρ τῆς ὑποστάσεως αὐτοῦ. Distinct vocabulary (χαρακτήρ, ὑπόστασις).
- **Eph 1:20–23** — exaltation parallel; ὑπερύψωσεν cognates may recur and should match (เทิดทูน...สูงสุด).
- **1 Pet 3:18–22** — ζῳοποιηθείς + cosmic ascent.
- **Rev 5:9–14** — heavenly-court worship of the Lamb; LXX-divine-name imagery picks up Phil 2:10–11.
