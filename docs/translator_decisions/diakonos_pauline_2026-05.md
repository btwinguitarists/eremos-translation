# διάκονος → ผู้รับใช้ (Pauline default; preserves NT-fluidity)

**Scope:** All Pauline διάκονος / διακονία / διακονέω occurrences when the referent is a human (not Christ). Christ-as-διάκονος (Rom 15:8) follows the same default. The female-deaconess question at 1 Tim 3:11 is **explicitly deferred** to the Pastorals end-of-book audit; this lock preserves the option-space.

**Decided:** 2026-05-02 during the Romans end-of-book audit, after external AI review (Gemini CONCERN — wants doc; ChatGPT FINE — same disposition: lock + write doc). Both AIs converge on the same action even though their verdicts differ in framing.

| Eph 6:21 (forward) | πιστὸς διάκονος ἐν κυρίῳ | Tychicus (co-worker / messenger διάκονος) | **ผู้รับใช้** |

The Pauline lemma covers Christ + apostles + co-workers + Phoebe + the false super-apostles + Pastoral-διάκονοι. **No single English or Thai term carries this full range.** ผู้รับใช้ ("servant" / "one who serves") is the broadest Thai term that fits all referents, preserving the Pauline-fluidity.

## The rule

### Default rendering

| Greek form | Thai rendering |
|---|---|
| διάκονος (substantive, generic-servant; any Pauline referent) | **ผู้รับใช้** |
| διάκονος + genitive of community/object (διάκονος τῆς ἐκκλησίας / περιτομῆς / Χριστοῦ / θεοῦ) | **ผู้รับใช้ของ + [community/object]** |
| διακονία (the activity / ministry / service) | **การรับใช้** OR contextually **พันธกิจ** ("ministry" — when the referent is a specific apostolic-mission, e.g., Rom 11:13 ἡ διακονία μου) |
| διακονέω (verbal — "serve / minister") | **รับใช้** OR **ปรนนิบัติ** (the latter when the action is concrete-bodily-service like Acts 6:2 table-service) |
| διάκονοι in the 1 Tim 3:8–13 office context | **deferred** — see "Pastoral epistles" below |

### Why ผู้รับใช้ (and not มัคนายก)

- **มัคนายก** is the modern Thai church-polity term for "deacon" as a formal office. Using it for Pauline διάκονος would import the post-NT institutional category onto the NT lemma, foreclosing the historical-fluidity question.
- **ผู้รับใช้** ("one who serves / servant") is the broadest Thai term covering generic-servanthood + apostolic-mission-service + specific-local-church-function — exactly the range Pauline διάκονος has.
- **The trajectory question Gemini raised is real:** when Eph 6:21 + Col 1:7 + 1 Tim 3:8 ship, all three need consistent treatment. ผู้รับใช้ keeps the corpus internally-consistent without pre-committing on church-polity questions.

### Why the female-deaconess question is deferred

Rom 16:1 (Phoebe as διάκονον τῆς ἐκκλησίας) and 1 Tim 3:11 (γυναῖκας in the διάκονοι-section) together form the canonical female-leadership-in-διάκονος-office question. Eremos has not yet translated 1 Tim. Pre-committing on Phoebe's office would lock the 1 Tim 3:11 reading before the 1 Tim audit can address it.

This lock therefore:
1. Renders Rom 16:1 Phoebe as **ผู้รับใช้** (preserving fluidity)
2. Defers the 1 Tim 3:11 question to the Pastorals end-of-book audit
3. **Does not** read Rom 16:1 as endorsing OR denying a formal female-deaconess office — the Greek's διάκονος-fluidity is preserved in Thai

### Royal vs plain register

- διάκονος + Christ-as-referent (Rom 15:8) — **plain register ผู้รับใช้**, NOT royal. The διάκονος-attribution is a humbling-self-statement (cf. Phil 2:7 μορφὴν δούλου λαβών — same humbling-cluster). Adding ทรง- to διάκονος would invert Paul's rhetorical force.
- διάκονος + apostolic / human referent — plain register ผู้รับใช้.
- διακονέω verbal action by Christ — context-dependent; usually plain register since the verb foregrounds the humbling-service-action (e.g., Mark 10:45 διακονῆσαι / διακονηθῆναι).

### Pastoral epistles (1-2 Tim, Titus)

The Pastorals introduce a more-formalized ecclesial structure (ἐπίσκοπος + διάκονος in apparent office-pairing at 1 Tim 3 + Phil 1:1; cf. Titus 1:5–9 πρεσβύτερος list). When the Pastorals ship:

1. **1 Tim 3:8–13** — the διάκονος "qualifications" passage. Three options: (a) preserve **ผู้รับใช้** (continuity with this lock; reader fills the office-vs-fluidity blank from context); (b) shift to formal **มัคนายก** (locks office-reading); (c) hybrid (ผู้รับใช้ in main text + footer-note acknowledging the office-reading possibility).
2. **1 Tim 3:11 γυναῖκας** — same options for the female-διάκονος / wives-of-deacons question.
3. **Phil 1:1 σὺν ἐπισκόποις καὶ διακόνοις** — the bishop-deacon pairing forces the office-reading question earlier than the Pastorals.

This lock does NOT resolve those questions. It preserves the option-space.

### James, Hebrews, 1 Peter

- Heb 1:14 (angels as διάκονοι "ministering spirits") — ผู้รับใช้
- 1 Pet 4:11 (διακονέω verbal) — รับใช้
- Apply same Pauline-lock principles when those books ship.
