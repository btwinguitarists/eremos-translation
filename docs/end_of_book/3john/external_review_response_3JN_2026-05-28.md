## Items in detail

### Item A — Diotrephes-conflict surfacing at 3JN 1:9–10
- **Gemini:** CONCERN. Risk of modern Thai church factions weaponizing "Diotrephes" as a personal-attack label is a documented cultural reality. Adding a summary to bound the application specifically to the fourfold structural cluster (slander + refusing hospitality + blocking + excommunicating) aligns with the historical-counterparts strategy used in 2 Timothy.
- **ChatGPT:** CONCERN. Same diagnosis. Specifies the summary should land at **v.10 not v.9** (because v.10 is where the full misconduct pattern is enumerated). Summary should state: issue is the combined pattern (not one isolated trait); John's intervention is personal/apostolic (not a generic congregational attack-template); modern application is structural/discernment-oriented (not name-calling).
- **Synthesized / Action:** Add v.10 thai_summary surfacing the fourfold cluster + the apostolic-authority context + the bounded-application caveat. Both AIs converge on Option B and on v.10 placement.

### Item B — φίλοι vs ἀδελφοί register split (1:15)
- **Gemini:** FINE. เพื่อน provides clean lexical contrast to พี่น้อง without introducing the overly literary/poetic baggage of มิตรสหาย. Don't over-theologize a personal greeting circle.
- **ChatGPT:** FINE. Same; explicitly **don't** add a v.15 thai_summary for the JHN 15 link — the link is real but v.15 doesn't foreground Last Discourse theology strongly enough to burden the reader-facing summary.
- **Synthesized / Action:** Lock เพื่อน. Keep JHN 15:14–15 connection in verse-level KD only; do not add v.15 summary.

### Item C — ὑπολαμβάνω → สนับสนุน (1:8)
- **Gemini:** FINE. สนับสนุน is the exact right semantic match for material/moral partnership; อุปถัมภ์ would introduce top-down patron/client dynamics that clash with the egalitarian συνεργοί ("fellow-workers") language in the same verse.
- **ChatGPT:** FINE. Same; explicitly warns against ค้ำจุน (over-literalizes "take up from below"; sounds like sustaining physically/emotionally rather than supporting itinerant gospel workers materially).
- **Synthesized / Action:** Lock สนับสนุน. Move the missionary-support semantic cluster (προπέμπω + ὑπολαμβάνω + συνεργός) to the corpus doc `propempo_missionary_sending_2026-05.md` rather than the verse-level KD.

### Item D — ὁ φιλοπρωτεύων → ผู้ที่ใฝ่หาความเป็นใหญ่ (1:9)
- **Gemini:** FINE. ใฝ่หา (active pursuit) captures the present-participle + φιλο- prefix; THSV's อยากเป็นใหญ่ makes the action too passive and internal.
- **ChatGPT:** FINE. Same; keep distinct from generic ambition vocabulary like ทะเยอทะยาน — this is a marked "love-to-be-first" Johannine formulation.
- **Synthesized / Action:** Lock ผู้ที่ใฝ่หาความเป็นใหญ่.

### Item E — "the Name" absolute Christological reference (1:7)
- **Gemini:** CONCERN. Royal-prefix พระนาม marks divine status, but the absolute use of "the Name" as early-Christian shorthand for Christ is easily lost on modern readers (who may read it as "God's reputation"). Add v.7 thai_summary identifying "the Name" as Christ's, **and cascade this exact surfacing strategy back to Acts 5:41** for corpus-level consistency.
- **ChatGPT:** CONCERN. Same. Add v.7 thai_summary explaining พระนาม = พระนามของพระคริสต์ shorthand. **Cascade a parallel summary or note to Acts 5:41** (closest corpus precedent). Keep verse text as `เพื่อพระนาม`; don't expand inline.
- **Synthesized / Action:** Add v.7 thai_summary; cascade to Acts 5:41 (both AIs explicitly converge on the cascade).
