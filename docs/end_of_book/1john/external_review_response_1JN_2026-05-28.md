## Items in detail

### Item A — ἀντίχριστος = ผู้ต่อต้านพระคริสต์
- **Gemini:** FINE. Literal Thai rendering preserves Greek morphology (anti + Christ) across 2:18, 2:22, 4:3; avoids dispensational baggage of transliterated อันติคริสต์.
- **ChatGPT:** FINE. Right Johannine default; explicitly distinguishes from ψευδόχριστος → พระคริสต์เทียมเท็จ and Revelation 13 beast language. Minor: if 2:18 summary reads "ปฏิญาณการต่อต้านพระคริสต์," revise to "ปรากฏการณ์การต่อต้าน..." or "รูปแบบการต่อต้าน..."
- **Synthesized / Action:** Lock as-is. Check 2:18 thai_summary against ChatGPT's copyedit flag.

### Item B — ἱλασμός / ἱλαστήριον bifurcation
- **Gemini:** CONCERN. เครื่องบูชาไถ่บาป works for 1JN 2:2, 4:10, ROM 3:25 — but applying the same Thai to HEB 9:5 creates a semantic collision (Hebrews 9:5 is the physical mercy-seat / ark cover, not a sacrificial instrument).
- **ChatGPT:** CONCERN. Same conclusion; suggests HEB 9:5 → object-noun (พระที่นั่งกรุณา or ฝาพระกรุณา), pending Thai-reviewer preference.
- **Synthesized / Action:** Lock 1JN + ROM at เครื่องบูชาไถ่บาป; split HEB 9:5 to physical-object surface. Write `hilasterion_hilasmos_bifurcation_2026-05.md`.
- **Cross-book agreement:** HEB Item F independently locks ROM 3:25 = เครื่องบูชาไถ่บาป and HEB 9:5 = พระที่นั่งกรุณา.

- **Synthesized / Action:** Lock for Johannine corpus. Write `johannine_menos_2026-05.md` with the Revelation exception baked in.

### Item E — 5:18 perfect vs aorist γεννάω distinction
- **Gemini:** FINE. Using royal prefix ทรง- for the divine aorist participle vs plain ได้- for the believer's perfect is a masterstroke of optimal equivalence. Write `johannine_begotten_trinity_2026-05.md`.
- **ChatGPT:** CONCERN — solution is correct, but the *rule formulation* matters: JHN 1:13 has an aorist passive for believers too, so the rule must be **referent-based, not tense-form-based**. Don't make aorist tense alone trigger ทรง-.
- **Synthesized / Action:** Lock 1JN 5:18 verse text as-is. Write the unified Johannine birth doc but **formulate the rule by referent (Christ vs believer), not tense form**. This is a clarification of ChatGPT's, not a real divergence on the verse outcome.
