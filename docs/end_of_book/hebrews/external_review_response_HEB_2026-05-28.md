## Items in detail

### Item A — HEB 1:8 ὁ θεός vocative
- **Gemini:** FINE. Contextual frame πρὸς δὲ τὸν υἱόν (v.8) demands direct address; Hellenistic Greek regularly uses articular nominative for vocative. ข้าแต่พระเจ้า is grammatically and contextually secure. **Don't** add a footer for the "God is your throne" alternative — violates evangelical-Protestant alignment.
- **ChatGPT:** FINE. Same. The "God is your throne" reading is syntactically possible but contextually weak (awkward metaphor; blunts Hebrews 1's Son-over-angels argument). If a note is ever added, it should be neutral textual/translation, not name a sectarian alternative.
- **Synthesized / Action:** Lock as-is. No footer.

### Item B — HEB 6:4–6 apostasy + ἀνασταυρόω
- **Gemini:** FINE. Maintain textual ambiguity; "again" reading (ซ้ำอีก) aligns with ESV/NIV/CSB and historic Protestant Hebrews-6 exegesis.
- **ChatGPT:** FINE. Same. The surrounding πάλιν ... εἰς μετάνοιαν already creates the renewal-again frame.
- **Synthesized / Action:** Lock as-is. Keep ซ้ำอีก. No theological footer.

### Item C — HEB 10:5 LXX σῶμα vs MT אזנים
- **Gemini:** FINE. NT translation must follow the NT author's actual cited Greek (σῶμα at v.5 because v.10 develops the body-offering logic). Standard, principled NT-translation practice. Omit the footer to avoid textual-criticism scope creep.
- **ChatGPT:** FINE with **one bounded footer** permitted because this is a high-visibility, argument-relevant LXX/MT divergence: "Greek Ps 40:6 reads 'body'; the Hebrew text reads 'ears.'" Don't make this a general policy for every minor LXX/MT divergence.
- **Synthesized / Action:** Lock พระกาย. **Minor divergence on whether to add a single high-visibility footer here** — ChatGPT permits; Gemini opposes. Ben picks. Write/confirm corpus rule: NT authorial citation form takes priority in NT translation even when it materially diverges from MT-OT.

### Item D — HEB 4:12 ψυχή / πνεῦμα crux  (**proposed verse edit**)
- **Gemini:** CONCERN. Two different ad-hoc pairings for the same Greek doublet (HEB 4:12 จิตวิญญาณ/วิญญาณ vs 1Th 5:23 จิตใจ/วิญญาณ) introduces inconsistency. Both verses deal with the piercing/division/totality of the human inner life; should ideally share a unified Thai pairing.
- **ChatGPT:** CONCERN. Exception itself is legitimate, but current HEB 4:12 จิตวิญญาณและวิญญาณ creates **avoidable overlap** because จิตวิญญาณ already contains วิญญาณ — making the two-member Greek distinction less clear in Thai. The 1Th 5:23 exception using จิตใจ / วิญญาณ is cleaner. **Spot-revise HEB 4:12 to จิตใจและวิญญาณ unless a Thai reviewer strongly objects.**
- **Synthesized / Action:** **PROPOSED VERSE EDIT** — HEB 4:12 จิตวิญญาณและวิญญาณ → จิตใจและวิญญาณ. Both AIs converge; ChatGPT's reasoning (overlap of จิตวิญญาณ ⊃ วิญญาณ blunts the distinction) is decisive. Then amend `psyche_vs_pneuma_anthropological_2026-04.md` with two explicit exceptions: 1Th 5:23 (trichotomy prayer: วิญญาณ / จิตใจ / กาย), HEB 4:12 (Word-piercing doublet: จิตใจ / วิญญาณ).

### Item E — HEB 12:14 ἁγιασμός
- **Gemini:** FINE. Forcing rigid process vs state via การ- / ความ- often results in unnatural Thai. ความบริสุทธิ์ flows naturally and conveys the required holiness. Revise the proposed `hagiasmos_hagiosune_2026-04.md` to allow both renderings context-dependent.
- **ChatGPT:** FINE. Same. The proposed hard lock ἁγιασμός = การชำระให้บริสุทธิ์ is too rigid. HEB 12:14 reads naturally as the pursued condition/quality of holiness, not an abstract theological process.
- **Synthesized / Action:** Lock ความบริสุทธิ์ at HEB 12:14. Revise the doc to allow both: การชำระให้บริสุทธิ์ where process foregrounded; ความบริสุทธิ์ where state/result/quality foregrounded.

### Item F — Forward ROM 3:25 ἱλαστήριον
- **Gemini:** FINE. Pauline theology in ROM 3:25 shifts focus to instrumental, propitiatory sacrifice rather than physical tabernacle furniture. เครื่องบูชาไถ่บาป aligns with mainstream evangelical Protestant exegesis and Thai reader expectations.
- **ChatGPT:** FINE. Same. HEB 9:5 is the physical mercy-seat (พระที่นั่งกรุณา); ROM 3:25 is Pauline atonement (เครื่องบูชาไถ่บาป). Don't use เครื่องลบล้างบาป as default — tilts too narrowly toward expiation, loses propitiatory force.
- **Synthesized / Action:** Lock ROM 3:25 = เครื่องบูชาไถ่บาป; HEB 9:5 stays พระที่นั่งกรุณา. Write the lock doc.
