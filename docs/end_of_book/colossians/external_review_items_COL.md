## Item A — The Colossians Christ-Hymn (1:15–20) cosmic-Christ vocabulary cluster

**The pattern:** COL 1:15–20 is the most theologically load-bearing passage in the book — the SECOND Christological hymn the Thai project has handled (after PHP 2:5–11), but with a completely different vocabulary register: cosmic-creator-and-reconciler instead of kenotic-incarnational. The translator preserved the Pauline lemma-pair distinctions while explicitly NOT mechanically-importing PHP 2 vocabulary (per the existing `christ_hymn_kenosis_2026-05.md` doc's stated principle).

| Greek | Thai | Editorial principle |
|---|---|---|
| **εἰκὼν** τοῦ θεοῦ τοῦ ἀοράτου (1:15) | **ฉายา**ของพระเจ้าผู้ที่ตามนุษย์ไม่อาจเห็นได้ | εἰκών = perfect-revelation-image (Son perfectly reveals Father), NOT photographic-likeness. |
| **πρωτότοκος** πάσης κτίσεως (1:15) | **บุตรหัวปีเหนือสิ่งที่ทรงสร้างทั้งสิ้น** | Uses **เหนือ** ("over") not **ของ** ("of") — preserves rank-comparative, blocks the Arian / JW first-created reading. See Item B for separate review. |
| ἐν αὐτῷ **ἐκτίσθη τὰ πάντα** ... δι' αὐτοῦ καὶ εἰς αὐτὸν ἔκτισται (1:16) | ในพระองค์ ทุกสิ่งได้ทรงถูกสร้างขึ้น ... โดยทางพระองค์และเพื่อพระองค์ | Triple-prepositional Christological frame (ἐν / διά / εἰς). Christ as sphere + agent + goal of creation. |
| αὐτός ἐστιν **πρὸ πάντων** καὶ **τὰ πάντα ἐν αὐτῷ συνέστηκεν** (1:17) | พระองค์**ทรงดำรงอยู่ก่อนสรรพสิ่ง** และในพระองค์ ทุกสิ่ง**ดำรงอยู่ด้วยกัน** | πρό = temporal pre-existence; συνέστηκεν perfect-tense = continuing-providential-cohesion. |
| **πρωτότοκος** ἐκ τῶν νεκρῶν (1:18) | **บุตรหัวปี**จากในหมู่ผู้ตาย | Second πρωτότοκος brackets the hymn (creation + resurrection). Identical Thai rendering preserves the structural-parallel. |
| αὐτός ... **πρωτεύων** (1:18c) | พระองค์เอง ... **ทรงเป็นที่หนึ่ง**ในทุกสิ่ง | NT-HAPAX πρωτεύω → **ทรงเป็นที่หนึ่ง**. Summarizes both halves of the hymn. |
| ἐν αὐτῷ εὐδόκησεν **πᾶν τὸ πλήρωμα** κατοικῆσาι (1:19) | พระเจ้าทรงพอพระทัยที่จะให้**ความบริบูรณ์ทั้งสิ้น**สถิตอยู่ในพระองค์ | πλήρωμα → **ความบริบูรณ์ทั้งสิ้น** (vs. Colossian-heresy distributing divinity across emanations). |
| **ἀποκαταλλάξαι τὰ πάντα** εἰς αὐτόν (1:20a) | ให้**ทุกสิ่งคืนดี**กับพระองค์เอง | Intensified-compound ἀποκαταλλάσσω → **ให้ ... คืนดี**. Cosmic scope of reconciliation preserved. |
| **εἰρηνοποιήσας διὰ τοῦ αἵματος τοῦ σταυροῦ** αὐτοῦ (1:20b) | ทรงทำให้เกิด**สันติสุขผ่านโลหิตที่หลั่งบนกางเขน**ของพระองค์ | NT-HAPAX εἰρηνοποιέω → **ทำให้เกิดสันติสุข**. Cross-blood as historical-particular means with cosmic-universal effect. |

**The cluster tests `christ_hymn_kenosis_2026-05.md`'s "validate the lock without mechanically importing PHP 2 vocabulary" principle.** No fresh corpus doc exists for the COL hymn yet; the audit recommends amending the existing kenosis doc with a COL 1:15–20 sub-section.

## Item B — πρωτότοκος (1:15) — anti-Arian rendering with **เหนือ**

**The textual decision:** COL 1:15 is the single most-attacked verse in the book in cult-vs-orthodox debates: πρωτότοκος πάσης κτίσεως. Two readings dominate:

- **(a) Rank-comparative** ("firstborn-OVER-all-creation, supreme over all created things") — *currently chosen.* πρωτότοκος is a Hebrew-Bible idiom for rank/preeminence (cf. Ps 89:27 LXX πρωτότοκον θήσομαι αὐτόν "I will make him firstborn"). Christ is the supreme-One over all creation, NOT a created being. The genitive πάσης κτίσεως is genitive-of-comparison/rank, NOT partitive.
- **(b) Partitive** ("firstborn-OF-all-creation, the first thing created among all creation") — the classic Arian / Jehovah's Witnesses NWT reading.

**Current Thai rendering** uses **บุตรหัวปีเหนือสิ่งที่ทรงสร้างทั้งสิ้น** — explicitly **เหนือ** ("over") instead of **ของ** ("of"):

- COL 1:15 GK: `ὅς ἐστιν εἰκὼν τοῦ θεοῦ τοῦ ἀοράτου, **πρωτότοκος πάσης κτίσεως**`
  → TH: `พระบุตรทรงเป็นฉายาของพระเจ้าผู้ที่ตามนุษย์ไม่อาจเห็นได้ ทรงเป็น**บุตรหัวปีเหนือสิ่งที่ทรงสร้างทั้งสิ้น**`

The 1:15 KD names the choice + cites v.16 (Christ-as-creator) as confirmation that the partitive-Arian reading is incoherent.

**Editorial significance:** Mainstream evangelical English Bibles (NIV / ESV / CSB / BSB) all preserve the literal "firstborn of all creation" + rely on context + reader-knowledge to block the Arian reading. The Thai project's choice to encode the rank-reading **lexically** (with **เหนือ**) is **MORE explicit than any of the major English Bibles** — defensible CC0 evangelical move (lexically blocks cult mis-readings) but does add an interpretive step beyond the bare Greek.

**Forward-pertinent:** The same πρωτότοκος-of-X recurs at COL 1:18 (ἐκ τῶν νεκρῶν → **บุตรหัวปีจากในหมู่ผู้ตาย**), Rom 8:29 (ἐν πολλοῖς ἀδελφοῖς), Heb 1:6 (Christ-brought-into-the-world), Heb 12:23 (πρωτοτόκων plural — believers), and Rev 1:5 (τῶν νεκρῶν).

**Two questions:**
1. Is **บุตรหัวปีเหนือสิ่งที่ทรงสร้างทั้งสิ้น** (with **เหนือ** "over") the right rendering for an evangelical-Protestant CC0 Thai Bible — going one step more-explicit than NIV / ESV / CSB / BSB to lexically block the Arian reading? Or should the rendering be more literally **บุตรหัวปีของสิ่งที่ทรงสร้างทั้งสิ้น** (with **ของ** "of") + rely on context + a footer-note?
2. Should the project lock the **บุตรหัวปี + เหนือ-vs-ของ** sense-distinction by writing a brief `prototokos_pauline_2026-05.md` doc (rank-comparative-of-creation: **เหนือ**; partitive-of-the-dead: **จากในหมู่**; relational-of-believers: **ในหมู่**) before Heb 1:6 + Rev 1:5 ship?

## Item C — στοιχεῖα τοῦ κόσμου (2:8, 2:20) — basic-principles vs elemental-spirits, the long-deferred GAL question

**The pattern:** The Galatians end-of-book audit (§4, 2026-04-30) flagged this as a corpus-priority lift recommendation: "Recommend new doc `docs/translator_decisions/stoicheia_tou_kosmou_2026-04.md` to lock หลักการ as the corpus-wide rendering ... Important pre-Col 2 handoff." **The doc was never written.** Now COL has shipped using the GAL precedent (**หลักการพื้นฐานของโลก**), and the question recurs in a context where the chapter's own internal evidence may lean differently than GAL did.

Three exegetical readings:
1. **Basic principles** — elementary religious systems (Jewish-law and pagan ritual alike). BSB / ESV / NIV / CSB majority. *Currently chosen at COL 2:8 + 2:20 per GAL precedent.*
2. **Physical elements** — the four elements of Greek philosophy.
3. **Elemental spirits** — personified spiritual powers (angelic beings) governing pre-Christian humanity. Increasingly favored in apocalyptic-Paul readings; argued more strongly for COL 2 by some commentators.

- COL 2:8 GK: `διὰ τῆς φιλοσοφίας καὶ κενῆς ἀπάτης κατὰ τὴν παράδοσιν τῶν ἀνθρώπων, **κατὰ τὰ στοιχεῖα τοῦ κόσμου** καὶ οὐ κατὰ Χριστόν`
  → TH: `ด้วยปรัชญาและการล่อลวงอันว่างเปล่า ตามประเพณีของมนุษย์ **ตามหลักการพื้นฐานของโลก** และไม่ตามพระคริสต์`
- COL 2:20 GK: `εἰ ἀπεθάνετε σὺν Χριστῷ **ἀπὸ τῶν στοιχείων τοῦ κόσμου**, τί ὡς ζῶντες ἐν κόσμῳ δογματίζεσθε`
  → TH: `ถ้าพวกท่านได้ตายร่วมกับพระคริสต์เพื่อพ้นจาก**หลักการพื้นฐานของโลก**แล้ว เหตุใดพวกท่านจึงยังยอมอยู่ใต้ข้อบังคับเหมือนกับยังมีชีวิตอยู่ในโลกนี้`

**The polemical context that may favor reading (3) for COL specifically:** chapter 2 is densely-elemental — 2:10 (Christ is head over πάσης ἀρχῆς καὶ ἐξουσίας — every spiritual-rule-and-authority); 2:15 (Christ disarmed-and-paraded τὰς ἀρχὰς καὶ τὰς ἐξουσίας); 2:18 (the false-teachers practice θρησκείᾳ τῶν ἀγγέλων — angel-worship — see Item D); 2:20 immediately after recapitulating Christ-vs-the-στοιχεῖα. The chapter's internal evidence leans toward elemental-spirits more strongly than GAL 4:3–9 did.

**Two questions:**
1. Is **หลักการพื้นฐานของโลก** (option 1: basic-principles) the right Pauline-corpus default for COL 2:8 + 2:20 — preserving corpus-consistency with already-shipped GAL 4:3, 4:9 — even though the immediate COL 2 context (anti-cosmic-powers polemic, angel-worship, defeated ἀρχαὶ καὶ ἐξουσίαι) leans toward reading (3) "elemental spirits"?
2. Either way, should the project finally write the GAL-recommended corpus doc as `stoicheia_tou_kosmou_2026-05.md` (locking the chosen reading + naming the alternatives + citing both GAL 4:3 and COL 2:8 as locking precedents)?

## Item D — θρησκεία τῶν ἀγγέλων (2:18) — subjective vs objective genitive

**The textual decision:** COL 2:18 contains a famous interpretive-crux genitive that determines the polemical-target of Paul's critique:

- COL 2:18 GK: `μηδεὶς ὑμᾶς καταβραβευέτω θέλων ἐν ταπεινοφροσύνῃ καὶ **θρησκείᾳ τῶν ἀγγέλων**, ἃ ἑόρακεν ἐμβατεύων, εἰκῇ φυσιούμενος ὑπὸ τοῦ νοὸς τῆς σαρκὸς αὐτοῦ`
  → TH: `อย่าให้ผู้ใดตัดสิทธิ์รางวัลของพวกท่าน ผู้ที่หลงใหลในความถ่อมใจจอมปลอมและ**การนมัสการทูตสวรรค์** ดื่มด่ำกับสิ่งที่เขาได้เห็น ถูกพองขึ้นโดยไม่มีเหตุ ด้วยความคิดของจิตใจฝ่ายเนื้อหนังของเขาเอง`

Two readings hinge on the genitive's function:

| Reading | Claim | Polemical target |
|---|---|---|
| **(a) Objective genitive** ("worship OFFERED-TO angels") — *currently chosen* | The false-teachers worship angels themselves; Paul rebukes angel-veneration. KJV / NIV / ESV / CSB / BSB mainstream evangelical reading. | The Colossian-heresy includes pagan-style angel-veneration. |
| **(b) Subjective genitive** ("worship OFFERED-BY angels" — i.e., the visionary-mystic claims to join the angelic-heavenly-worship) | The false-teachers claim to participate-in / observe heavenly-worship through visionary-mystical experiences (cf. 2:18b ἃ ἑόρακεν ἐμβατεύων "entering-into what he has seen"). Increasingly favored in modern scholarship (Fred Francis 1962; Lincoln; Dunn). | The Colossian-heresy is mystical-Jewish-apocalyptic visionary-piety, not pagan angel-cult. |

**Current Thai rendering** **การนมัสการทูตสวรรค์** ("the worship-of angels") is the literal objective-genitive reading. The Thai genitive **ของ-construction** is ambiguous on subjective-vs-objective just as Greek is — but reader-default will resolve to (a) angel-worship without explicit context.

The 2:18 KD names the false-teachers' "angel-veneration" as the polemical target; the rendering aligns with reading (a). Reading (b) is supported contextually by 2:18b (ἃ ἑόρακεν ἐμβατεύων + φυσιούμενος "puffed-up by his fleshly mind") + 2:23 (ἐθελοθρησκίᾳ "self-imposed-worship").

**Two questions:**
1. Is reading (a) angel-veneration the right corpus-evangelical-default — accepting the slight-loss of academic-consensus reading (b) visionary-mystical-piety? The Thai rendering preserves the genitive ambiguity formally but Thai-reader-default resolves to (a) without explicit cue.
2. Should the project add a footer-note at 2:18 acknowledging the alternative academic reading (b) — or leave the rendering uncommented per RULES §0 evangelical-Protestant alignment + space-economy?

## Item E — ἀνταναπληρῶ + τὰ ὑστερήματα τῶν θλίψεων τοῦ Χριστοῦ (1:24)

**The textual decision:** COL 1:24 is probably the most-difficult-to-translate verse in the book theologically. The lemma ἀνταναπληρόω is a NT-HAPAX (compound: ἀντί + ἀνά + πληρόω = "fill-up-in-turn / fill-up-on-behalf-of"). The Greek phrase has been historically used as a Catholic proof-text for the merit-of-the-saints theology (the believer's sufferings supplement Christ's atonement) — and rejected by Reformation-evangelical scholarship as misreading the lemma.

- COL 1:24 GK: `Νῦν χαίρω ἐν τοῖς παθήμασιν ὑπὲρ ὑμῶν, καὶ **ἀνταναπληρῶ τὰ ὑστερήματα τῶν θλίψεων τοῦ Χριστοῦ** ἐν τῇ σαρκί μου ὑπὲρ τοῦ σώματος αὐτοῦ, ὅ ἐστιν ἡ ἐκκλησία`
  → TH: `บัดนี้ ข้าพเจ้าชื่นชมยินดีในความทุกข์ที่ทนเพื่อพวกท่าน และข้าพเจ้าก็**เติมเต็มในเนื้อหนังของข้าพเจ้า ในสิ่งที่ยังขาดอยู่ในความทุกข์ของพระคริสต์** เพื่อกายของพระองค์ คือ คริสตจักร`

The Thai preserves the literal "fill-up what is lacking in Christ's afflictions in my flesh." The 1:24 KD names the two-readings problem explicitly: "NOT 'Christ's-atonement was insufficient' — the lack refers to 'what Christ-intentionally-left for-his-disciples-to-suffer-as-his-body.'" The 1:24 thai_summary handles the orthodox-vs-Catholic-Mass-of-merits interpretive question explicitly. The orthodox interpretation rides on the thai_summary disambiguation rather than on the verse-translation itself.

**Question:** Is the literal **เติมเต็ม ... ในสิ่งที่ยังขาดอยู่ในความทุกข์ของพระคริสต์** + thai_summary disambiguation strategy the right approach for an evangelical-Protestant CC0 Thai Bible? Or should the verse rendering itself be more interpretively-explicit — e.g., **เติมเต็มความทุกข์ที่พระคริสต์ทรงประทานให้ข้าพเจ้าทนเพื่อกายของพระองค์** ("fill up the sufferings Christ allotted me to endure for his body") — which is doctrinally-orthodox-explicit but interpretively-stronger than the Greek itself? The Thai-reader-without-summary access is the relevant audience.

## Item F — The spiritual-beings 4-tier hierarchy (1:16, 2:10, 2:15)

**The pattern:** COL 1:16 lists four classes of spiritual-beings in a polemical-cosmological context (Paul's claim: ALL spiritual-beings are Christ's creatures, NOT co-equal-objects-of-veneration like the false-teachers' angel-worship). The translator chose **four distinct Thai terms** preserving the hierarchical-distinctions:

| Greek (1:16) | Thai | Sense |
|---|---|---|
| θρόνοι | **บัลลังก์** | thrones (literal — heavenly thrones / throne-occupying powers) |
| κυριότητες | **เทพผู้ครอง** | dominions / lordly-powers (divine-status spiritual-rulers) |
| ἀρχαί | **ทูตสวรรค์ผู้มีอำนาจ** | rulers (powerful angels — the lemma also = principalities / governing-spirits) |
| ἐξουσίαι | **ผู้ทรงสิทธิ** | authorities (those-with-rightful-power) |

- COL 1:16 GK: `ὅτι ἐν αὐτῷ ἐκτίσθη τὰ πάντα ἐν τοῖς οὐρανοῖς καὶ ἐπὶ τῆς γῆς, τὰ ὁρατὰ καὶ τὰ ἀόρατα, εἴτε **θρόνοι** εἴτε **κυριότητες** εἴτε **ἀρχαὶ** εἴτε **ἐξουσίαι**`
  → TH: `เพราะในพระองค์ ทุกสิ่งได้ทรงถูกสร้างขึ้น ทั้งในสวรรค์และบนแผ่นดินโลก ทั้งสิ่งที่มองเห็นได้และมองไม่เห็น ไม่ว่าจะเป็น**บัลลังก์**หรือ**เทพผู้ครอง** หรือ**ทูตสวรรค์ผู้มีอำนาจ**หรือ**ผู้ทรงสิทธิ**`

The cluster recurs **collapsed-to-pair** at 2:10 (πάσης ἀρχῆς καὶ ἐξουσίας → **ผู้ครองและผู้ทรงอำนาจทั้งสิ้น**) and 2:15 (τὰς ἀρχὰς καὶ τὰς ἐξουσίας → **ผู้ครองและผู้ทรงอำนาจ**). The collapsed-pair renderings are slightly-different (**ผู้ครอง** "ruler" not the longer **ทูตสวรรค์ผู้มีอำนาจ**) — context determines whether the angelic-specifically (1:16) or the generic-rulership (2:10, 2:15) sense is foregrounded.

**Forward-pertinent:** Eph 1:21 (πάσης ἀρχῆς καὶ ἐξουσίας καὶ δυνάμεως καὶ κυριότητος — three of the four COL terms recur + add δύναμις); Eph 3:10; Eph 6:12 (ἀρχάς + ἐξουσίας + κοσμοκράτορας τοῦ σκότους); 1 Pet 3:22; Rom 8:38–39.

**Two questions:**
1. Is the 4-tier rendering at 1:16 (บัลลังก์ / เทพผู้ครอง / ทูตสวรรค์ผู้มีอำนาจ / ผู้ทรงสิทธิ) the right register for Thai-evangelical readers — or is it too-baroque / too-over-explicit for the reader's ear? Alternative: simplify to four-Thai-words matching English's "thrones / dominions / rulers / authorities" lexical-distinctness?
2. Is the contextual-shift between 1:16 ἀρχαί → **ทูตสวรรค์ผู้มีอำนาจ** (angelic-specifically) and 2:10 / 2:15 ἀρχαί → **ผู้ครอง** (generic-rulership) principled — same lemma rendered differently in different polemical contexts? Or should the project lock a single rendering for the lemma cluster + write a brief `spiritual_beings_hierarchy_2026-05.md` doc before Eph 1:21 + 6:12 ship?

## Item G — The Pauline household-code / Haustafel (3:18–4:1)

**The pattern:** COL 3:18–4:1 is the project's first Pauline-Haustafel (household-code / Greek-Roman household-ethics-tradition Christianized). Six imperatives across six relational-pairs:

| Verse | Greek | Thai |
|---|---|---|
| 3:18 | Αἱ γυναῖκες, **ὑποτάσσεσθε** τοῖς ἀνδράσιν, ὡς ἀνῆκεν ἐν κυρίῳ | ภรรยาทั้งหลาย จง**ยอมอยู่ใต้บังคับ**สามีของตน เพราะเป็นการเหมาะสมในองค์พระผู้เป็นเจ้า |
| 3:19 | οἱ ἄνδρες, **ἀγαπᾶτε** τὰς γυναῖκας καὶ μὴ πικραίνεσθε πρὸς αὐτάς | สามีทั้งหลาย จง**รัก**ภรรยาของตน และอย่าโกรธขึ้งใส่พวกเธอ |
| 3:20 | Τὰ τέκνα, **ὑπακούετε** τοῖς γονεῦσιν κατὰ πάντα | บุตรทั้งหลาย จง**เชื่อฟัง**บิดามารดาของตนในทุกสิ่ง |
| 3:21 | οἱ πατέρες, **μὴ ἐρεθίζετε** τὰ τέκνα ὑμῶν, ἵνα μὴ ἀθυμῶσιν | บิดาทั้งหลาย อย่า**ยั่วยุ**บุตรของท่าน เพื่อพวกเขาจะไม่ท้อใจ |
| 3:22 | οἱ δοῦλοι, **ὑπακούετε** κατὰ πάντα τοῖς κατὰ σάρκα κυρίοις | ทาสทั้งหลาย จง**เชื่อฟัง**นายของตนตามฝ่ายเนื้อหนังในทุกสิ่ง |
| 4:1 | οἱ κύριοι, τὸ δίκαιον καὶ τὴν ἰσότητα τοῖς δούλοις **παρέχεσθε** | นายทั้งหลาย จง**ปฏิบัติต่อ**ทาสของท่านอย่างยุติธรรมและเสมอภาค |

The 3:18 KD names the principle: "Reciprocal-Pauline pattern: wives submit + husbands love (v.19) — both governed by 'in the Lord.' Note: the imperative is middle-voice 'submit-yourselves,' not passive 'be-submitted.'" The corpus-precedent for ὑποτάσσω → **ยอมอยู่ใต้บังคับ** matches LUK 10:17, 1Co 14:32; for ὑπακούω → **เชื่อฟัง** matches Eph 6:1.

**Editorial significance:** The Pauline-Haustafel is among the most-debated passages in modern Christian-ethics discourse — egalitarian-vs-complementarian debate; mutual-submission readings of Eph 5:21–33; reception of the slave-master code post-abolition (Thailand abolished slavery in 1905).
